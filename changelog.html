# Changelog

# 0.1.3 - 14 January, 2026

# New features

  • Add deploy and undeploy commands for updating and rolling back databases in production environments.
  • Add a pull command to parse an existing database into an ACA schema.
  • Fix the version issue of dependent packages.

# 0.1.1 - 20 February, 2025

# New features

  • Adding OMIT parameter for returning
  • Add the skipConfeldt parameter to the insert method

# 0.1.00 - 12 September, 2024

# New features

  • Remote functions support file path namespace
  • Support writing schema to multiple files
  • Class decorator changed from "$" namespace to "C" namespace, class member decorator changed from "$_" namespace to "M" namespace

# 0.0.93 - 05 August, 2024

# Bug fixes

  • Change the npm package Bettersqlite3 to sqlite3

# 0.0.92 - 03 August, 2023

# Bug fixes

  • Fix a bug where the default values for newly added fields in the schema do not take effect
  • Fix that the return value can return null
  • Aggregation function count adds '*' field query

# New features

  • Add View Api

# 0.0.91 - 14 June, 2023

# New features
