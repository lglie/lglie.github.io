Aca.ts

Auto synchronize database schema; Auto generate database APIs; Auto generate the backend functions and database APIs into the frontend APIs, implement backend code development together with the frontend

Get Started →

Typescript API

The generated frontend and backend APIs are ts, with friendly code prompts, and can be developed without looking at documents

Frontend

Automatically generate APIs to access the backend without defining routes and communicating with backend developers

Backend

The backend code is developed on the frontend as required, and can be easily migrated to the backend after development

Framework independent

Fits any frontend and backend JS frameworks

Database

Support transaction, mapping and table relation constraint

Namespace

Both frontend and backend database APIs and functions support namespaces
